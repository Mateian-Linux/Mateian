Get Started in 4 Steps

1

Download

Get the ISO file from the link above. We support direct download and BitTorrent.

2

Flash

Use BalenaEtcher or Rufus to write the Mateian ISO to a USB stick (4GB+).

3

Boot

Insert USB, restart, and select "Install Mateian" from the GRUB menu shown above.

4

Install

Follow the Calamares installer instructions. You'll be ready in ~15 minutes.
